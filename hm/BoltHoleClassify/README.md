### 功能
+ 螺栓孔-螺栓尺寸分类, 并赋予梁单元

-----------------
### 控制参数
+ 名称前缀

-----------------
### 操作
1. 设置控制参数
2. 计算

-----------------
## 原理
1. 根据孔周围单元elem，获取bar单元及rbe2单元
2. 根据根据孔上3点坐标获取圆的半径，选择bar两端孔最小半径的孔径作为螺栓半径
3. 根据 int((r+0.05)*2)/2 定义Beam的半径尺寸
4. 对bar进行分类并创建BEAM单元
